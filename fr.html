Politique de Confidentialité

Dernière mise à jour : 25 février 2026

Développeur : GaOJoLab

Contact : gaojolab@gmail.com

Vue d'ensemble

Verseo ("l'Application") est une application de compagnon spirituel 100% hors ligne qui respecte votre vie privée. Cette politique explique quelles données sont collectées et comment elles sont utilisées.

Collecte de données

Informations personnelles

Nous NE collectons PAS :

  • Noms, adresses email ou numéros de téléphone
  • Comptes utilisateur ou identifiants de connexion
  • Données de localisation
  • Coordonnées
  • Informations financières
  • Vos recherches personnelles ou contenu d'étude biblique

Données analytiques (non personnelles)

Verseo utilise Firebase Analytics pour améliorer l'expérience de l'application. Les données anonymes suivantes sont collectées :

Informations collectées :

  • Données d'interaction produit : Quelles fonctionnalités vous utilisez (ex : recherche, verset du jour, thèmes d'exploration)
  • Données d'utilisation : Durée des sessions, écrans consultés, fréquence d'utilisation des fonctionnalités
  • Identifiant d'appareil : Firebase App Instance ID anonyme (non lié à votre identité)

Important :

  • ✅ Toutes les données analytiques sont anonymes et non identifiables
  • ✅ Les données ne sont PAS liées à votre identité
  • ✅ Les données ne sont PAS utilisées pour du tracking entre applications ou sites web
  • ✅ Les données sont utilisées UNIQUEMENT à des fins analytiques pour améliorer l'application

Achats intégrés et abonnements

Verseo propose les options d'achat suivantes :

  • Essai gratuit de 3 jours suivi d'un abonnement mensuel
  • Accès à vie avec un paiement unique

Traitement des paiements :

  • Tous les paiements sont traités par Apple via l'App Store
  • Nous ne collectons ni ne stockons AUCUNE information de paiement (carte bancaire, adresse de facturation, etc.)
  • Apple gère toutes les données de transaction conformément à sa Politique de confidentialité Apple

Gestion des abonnements :

  • Les abonnements sont gérés via votre identifiant Apple
  • Aucun compte utilisateur n'est requis dans l'application
  • Votre statut d'abonnement est vérifié localement à l'aide de la validation de reçu sécurisée d'Apple
  • Annulez à tout moment via Réglages iOS > [Votre Nom] > Abonnements

Données stockées localement :

  • Statut d'abonnement (essai gratuit, abonnement actif ou accès à vie)
  • Date d'expiration de l'essai (pour appliquer l'essai gratuit de 3 jours)
  • Ces données sont stockées uniquement sur votre appareil et ne sont pas transmises à nos serveurs

Ce que nous ne collectons PAS :

  • Moyens de paiement ou informations de facturation
  • Historique des transactions (géré uniquement par Apple)
  • Informations personnelles pour la création de compte

Stockage local des données

Les données suivantes sont stockées uniquement sur votre appareil et ne sont jamais transmises :

  • Vos versets bibliques favoris
  • Votre historique de recherche
  • Votre progression dans le jardin spirituel
  • Vos préférences d'application (langue, thème)
  • Votre statut de complétion de l'onboarding

Apple Intelligence (iOS 26+)

Sur les appareils compatibles (iPhone 15 Pro ou plus récent avec iOS 26+), Verseo utilise Apple Foundation Model pour générer des réponses empathiques. Tout le traitement IA se fait entièrement sur votre appareil sans aucune transmission de données vers des serveurs externes.

Services tiers

Firebase Analytics (Google)

Nous utilisons Firebase Analytics pour comprendre comment les utilisateurs interagissent avec l'application. Firebase peut collecter des informations anonymes sur l'appareil et l'utilisation. Pour plus d'informations :

  • Politique de confidentialité Firebase
  • Politique de confidentialité Google

APIs Apple

L'application utilise les APIs Apple standard :

  • UserDefaults : Pour stocker les préférences localement
  • CoreData : Pour stocker les favoris et la progression localement
  • CoreML : Pour le traitement IA sur l'appareil (aucune transmission de données)
  • StoreKit : Pour les achats intégrés et la gestion des abonnements (traité par Apple)

Sécurité des données

  • Toutes les données sont stockées localement sur votre appareil via les mécanismes sécurisés d'Apple
  • Aucune donnée utilisateur n'est transmise à des serveurs externes
  • Aucun réseau de tracking ou de publicité tiers n'est utilisé

Vos droits

Vous pouvez :

  • Supprimer toutes les données : Désinstaller l'application pour effacer toutes les données locales
  • Désactiver les analytics : Réglages iOS > Confidentialité et sécurité > Analyse et améliorations > Partager les analyses de l'iPhone (DÉSACTIVÉ)
  • Gérer les abonnements : Réglages iOS > [Votre Nom] > Abonnements
  • Demander des informations : Nous contacter à contact@verseo.app

Confidentialité des enfants

Verseo ne collecte pas sciemment de données auprès d'enfants de moins de 13 ans. L'application est classée 4+ et contient uniquement du contenu biblique adapté à tous les âges.

Modifications de cette politique

Nous pouvons mettre à jour cette politique périodiquement. La date "Dernière mise à jour" en haut indique la version la plus récente.
