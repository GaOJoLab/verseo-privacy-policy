Política de Privacidade

Última atualização: 25 de fevereiro de 2026

Desenvolvedor: GaOJoLab

Contato: gaojolab@gmail.com

Visão geral

Verseo ("o Aplicativo") é um aplicativo de companheiro espiritual 100% offline que respeita sua privacidade. Esta política explica quais dados são coletados e como são usados.

Coleta de dados

Informações pessoais

NÃO coletamos:

  • Nomes, endereços de e-mail ou números de telefone
  • Contas de usuário ou credenciais de login
  • Dados de localização
  • Informações de contato
  • Informações financeiras
  • Suas pesquisas pessoais ou conteúdo de estudo bíblico

Dados analíticos (não pessoais)

Verseo usa Firebase Analytics para melhorar a experiência do aplicativo. Os seguintes dados anônimos são coletados:

Informações coletadas:

  • Dados de interação do produto: Quais recursos você usa (ex: pesquisa, versículo do dia, temas de exploração)
  • Dados de uso: Duração das sessões, visualizações de tela, frequência de uso dos recursos
  • ID do dispositivo: Firebase App Instance ID anônimo (não vinculado à sua identidade)

Importante:

  • ✅ Todos os dados analíticos são anônimos e não identificáveis
  • ✅ Os dados NÃO estão vinculados à sua identidade
  • ✅ Os dados NÃO são usados para rastreamento entre aplicativos ou sites
  • ✅ Os dados são usados APENAS para fins analíticos para melhorar o aplicativo

Compras no aplicativo e assinaturas

Verseo oferece as seguintes opções de compra:

  • Teste gratuito de 3 dias seguido de uma assinatura mensal
  • Acesso vitalício com um pagamento único

Processamento de pagamentos:

  • Todos os pagamentos são processados pela Apple através da App Store
  • NÃO coletamos ou armazenamos nenhuma informação de pagamento (cartão de crédito, endereço de cobrança, etc.)
  • A Apple gerencia todos os dados de transações de acordo com sua Política de Privacidade da Apple

Gerenciamento de assinaturas:

  • As assinaturas são gerenciadas através do seu Apple ID
  • Nenhuma conta de usuário é necessária no aplicativo
  • Seu status de assinatura é verificado localmente usando a validação segura de recibo da Apple
  • Cancele a qualquer momento através de Configurações do iOS > [Seu Nome] > Assinaturas

Dados armazenados localmente:

  • Status de assinatura (teste gratuito, assinatura ativa ou acesso vitalício)
  • Data de expiração do período de teste (para aplicar o teste gratuito de 3 dias)
  • Esses dados são armazenados apenas no seu dispositivo e não são transmitidos para nossos servidores

O que NÃO coletamos:

  • Métodos de pagamento ou informações de cobrança
  • Histórico de transações (gerenciado apenas pela Apple)
  • Informações pessoais para criação de conta

Armazenamento local de dados

Os seguintes dados são armazenados apenas no seu dispositivo e nunca transmitidos:

  • Seus versículos bíblicos favoritos
  • Seu histórico de pesquisa
  • Seu progresso no jardim espiritual
  • Suas preferências do aplicativo (idioma, configurações de tema)
  • Seu status de conclusão da integração

Apple Intelligence (iOS 26+)

Em dispositivos compatíveis (iPhone 15 Pro ou mais recente com iOS 26+), Verseo usa Apple Foundation Model para gerar respostas empáticas. Todo o processamento de IA acontece inteiramente no seu dispositivo sem transmissão de dados para servidores externos.

Serviços de terceiros

Firebase Analytics (Google)

Usamos Firebase Analytics para entender como os usuários interagem com o aplicativo. O Firebase pode coletar informações anônimas do dispositivo e de uso. Para mais informações:

  • Política de Privacidade do Firebase
  • Política de Privacidade do Google

APIs da Apple

O aplicativo usa APIs padrão da Apple:

  • UserDefaults: Para armazenar preferências localmente
  • CoreData: Para armazenar favoritos e progresso localmente
  • CoreML: Para processamento de IA no dispositivo (sem transmissão de dados)
  • StoreKit: Para compras no aplicativo e gerenciamento de assinaturas (processado pela Apple)

Segurança de dados

  • Todos os dados são armazenados localmente no seu dispositivo usando mecanismos de armazenamento seguro da Apple
  • Nenhum dado do usuário é transmitido para servidores externos
  • Nenhuma rede de rastreamento ou publicidade de terceiros é usada

Seus direitos

Você pode:

  • Excluir todos os dados: Desinstale o aplicativo para remover todos os dados locais
  • Desativar analytics: Configurações do iOS > Privacidade e segurança > Análise e melhorias > Compartilhar análise do iPhone (DESATIVADO)
  • Gerenciar assinaturas: Configurações do iOS > [Seu Nome] > Assinaturas
  • Solicitar informações: Entre em contato conosco em contact@verseo.app

Privacidade infantil

Verseo não coleta intencionalmente dados de crianças menores de 13 anos. O aplicativo é classificado como 4+ e contém apenas conteúdo bíblico adequado para todas as idades.

Alterações nesta política

Podemos atualizar esta política periodicamente. A data "Última atualização" no topo indica a versão mais recente.

Resumo

Verseo é um aplicativo que prioriza a privacidade. Sua jornada espiritual, pesquisas e favoritos permanecem no seu dispositivo. Coletamos apenas análises anônimas para melhorar a experiência do aplicativo. Todos os pagamentos são processados com segurança pela Apple.
